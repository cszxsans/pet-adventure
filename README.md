# 星宠物语 (Pet Odyssey)

一款类似宝可梦/洛克王国的 2D 回合制宠物收集对战游戏，融合神兽之塔的消除棋盘玩法。

## 如何运行

直接用浏览器打开 `index.html` 即可开始游戏！

```bash
# Windows: 双击 index.html 或在浏览器地址栏输入
file:///D:/game_dev/pet-adventure/index.html
```

## 游戏玩法

### 核心机制

1. **消除棋盘** - 6×5 的元素球棋盘，点击选中一个球，再点击相邻球进行交换
2. **三消规则** - 3个或以上相同属性球连成一线即可消除
3. **行动点** - 每次消除获得行动点，用于释放宠物技能
4. **能量条** - 消除对应属性球可为我方宠物充能，满能量可放大招

### 属性系统

| 属性 | 克制 | 颜色 |
|------|------|------|
| 火 🔥 | 克草 | 红橙 |
| 水 💧 | 克火 | 青蓝 |
| 草 🌿 | 克水 | 绿色 |
| 电 ⚡ | 克暗 | 金黄 |
| 暗 🌑 | 克草 | 紫色 |

### 游戏流程

1. **主菜单** → 点击「开始冒险」
2. **世界地图** → 选择章节进入关卡
3. **战斗场景** → 消除棋盘 + 点击宠物施放技能
4. **胜利后** → 可选择捕捉野生宠物
5. **宠物图鉴** → 查看已收集的宠物
6. **我的队伍** → 管理6只出战宠物

### 捕捉系统

战斗胜利后出现野生宠物，可用三种捕捉球：
- **普通球** x10 - 基础捕捉率 30%
- **超级球** x3 - 捕捉率提升 50%
- **大师球** x1 - 捕捉率 90%

## 技术规格

- **引擎**: 纯 Canvas 2D 渲染
- **分辨率**: 720×1280 竖屏
- **存储**: localStorage 自动存档
- **音效**: Web Audio API 合成

## 操作说明

- **消除**: 点击选中球 → 点击相邻球交换
- **技能**: 点击我方宠物卡片施放技能
- **捕捉**: 胜利后选择捕捉球类型
- **返回**: 点击左下角返回按钮
