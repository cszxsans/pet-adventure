# 宝可梦类回合制宠物收集对战游戏 — 规格说明书

## 1. 项目概述

**项目名称**：星宠物语（Pet Odyssey）
**类型**：2D 回合制宠物收集 + 策略对战 + 塔防消除游戏
**核心玩法**：消除类棋盘（参考神兽之塔）+ 宠物捕捉培养 + 回合制对战
**目标用户**：休闲玩家，喜欢收集养成 + 轻度策略

---

## 2. 视觉与渲染规格

### 场景与主题
- **主色调**：暗紫夜空（#0d0b1e）配 金色/琥珀高光（#f5a623）
- **UI 风格**：卡片式 + 像素风宠物精灵，融合现代暗色 UI
- **字体**：Noto Sans SC（中文）、Press Start 2P（像素英文）

### 场景列表
1. **主菜单** — Logo + 三个按钮（开始冒险 / 图鉴 / 退出）
2. **世界地图** — 章节选择，5个地图章节，每章 5 关卡
3. **战斗场景** — 消除棋盘 + 宠物站位 + 对手宠物
4. **捕捉场景** — 野生宠物展示 + 捕捉动画
5. **宠物图鉴** — 收集所有宠物卡片
6. **宠物背包** — 管理队伍（最多6只）

### 棋盘设计（核心消除）
- 6 列 × 5 行格子（参考神兽之塔）
- 每格随机放置一种宠物元素球（5种属性 × 宠物形态）
- **拖拽交换**：点击选中一个球，再点击相邻球完成交换
- **三消规则**：横向或纵向 3 个及以上相同即消除
- **消除后**：上方球下落补位，顶层随机生成新球
- **Combo 计数**：连续消除叠加伤害倍率

## 3. 宠物系统

### 宠物数据（共20种，5属性）
| 属性 | 代表宠物 |
|------|---------|
| 火 | 小火鸦、烈焰狐、炽焰龙 |
| 水 | 碧波龟、蓝鳞鱼、激流海马 |
| 草 | 草芽兔、花仙鹿、巨树精灵 |
| 电 | 闪电鼠、雷角兽、暴风鹏 |
| 暗 | 暗影猫、幽冥狼、虚空领主 |

### 宠物属性
- **等级**：1~50 级，战斗获取经验升级
- **HP / ATK / DEF**：数值属性
- **技能**：每宠物 2 个（普攻 + 奥义）
- **稀有度**：★ ~ ★★★★★（白/绿/蓝/紫/金）
- **性格**：影响属性成长（10种性格）

### 捕捉系统
- 野生宠物每局随机生成（使用已解锁的宠物池）
- 捕捉率 = 基础 30% × (1 - 目标HP%×0.5) × 球加成
- 捕捉球有等级之分（普通/超级/大师）

---

## 4. 战斗系统

### 回合制对战
- 消除棋盘完成 → 产生行动点（每消除1组+1点，每Combo额外+1点）
- 行动点用于释放宠物技能（拖拽施放或点击技能按钮）
- 消除对应属性球 → 宠物获得额外能量（满能量可放大招）
- 敌方也有 AI，固定回合施放技能

### 伤害公式
```
伤害 = (ATK - 对方DEF×0.5) × 属性克制系数 × 技能倍率 × (0.9~1.1随机)
```
属性克制：火>草>水>火，电克暗，暗克草，草克水

### 关卡设计
| 章节 | 主题 | 怪物等级 |
|------|------|---------|
| 1 | 草原试炼 | 1-10 |
| 2 | 河流激战 | 10-20 |
| 3 | 火山秘境 | 20-30 |
| 4 | 雷电领域 | 30-40 |
| 5 | 虚空王座 | 40-50 |

## 5. 音效与动画

- **消除动画**：球缩放 + 发光 + 粒子爆炸
- **宠物攻击**：精灵左右冲刺 + 特效光效
- **捕捉动画**：球晃动 + 闪光 + 捕获成功/失败特效
- **背景音乐**：Web Audio API 合成 8-bit 风格 BGM
- **音效**：消除音、攻击音、升级音、捕捉音

---

## 6. 数据存储

- 所有数据存储在 `localStorage`
- 自动存档：每次战斗结束 / 捕捉 / 升级时
- 数据结构：
```json
{
  "player": { "name": "训练师", "coins": 1000, "capturedPets": [], "team": [] },
  "progress": { "chapter": 1, "stage": 1, "highestStage": 1 },
  "settings": { "music": true, "sfx": true }
}
```

---

## 7. 技术规格

- **渲染**：Canvas 2D（单文件 HTML，内联 JS/CSS）
- **分辨率**：720×1280 竖屏，Canvas 固定大小，CSS 等比缩放
- **帧率**：requestAnimationFrame，目标 60fps
- **状态管理**：简单状态机（menu/map/battle/capture/dict/backpack）
- **动画系统**：Tween 插值动画，支持队列播放
- **音效**：Web Audio API（OscillatorNode 合成）
